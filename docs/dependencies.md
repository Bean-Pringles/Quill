# Dependencies

NOTE: Dependencies needed for development have (dev) next to them. Developers will need all of them in this list, while users need the ones with (dev). To have all dependencies, you need the ones under `Both OSes` and your OS

## Both OSes

- cargo-eval (Install through command: `cargo install eval`) (dev)
- Rust/Cargo/Rustup (All of Rust's toolchain) (dev, needed if you want to compile the rust programs)
- Nim Compiler (All of Nim's toolchain) (dev, needed if you want to compile from src)
- UPX 
- Strip

## Windows

- Python (dev, needed if you want to run tests or generated Python programs)
- LLVM/Clang (most modern version)

## Linux

- Python3 (dev, needed if you want to run tests or generated Python programs)
- LLVM16 (Must be LLVM16)
